## ðŸ“Š Performance Metrics

### Key Metrics

**Impressions**
- Number of times ad was shown
- Measures reach

**Clicks**
- Number of times ad was clicked
- Measures interest

**CTR (Click-Through Rate)**
- Percentage of impressions that resulted in clicks
- Formula: (Clicks / Impressions) Ã— 100

**CPC (Cost Per Click)**
- Average cost for each click
- Formula: Spent / Clicks

**CPM (Cost Per Mille)**
- Cost per 1,000 impressions
- Formula: (Spent / Impressions) Ã— 1000

**CPE (Cost Per Engagement)**
- Average cost for each engagement
- Formula: Spent / Engagements

**CVR (Conversion Rate)**
- Percentage of clicks that converted
- Formula: (Conversions / Clicks) Ã— 100

**CPA (Cost Per Acquisition)**
- Average cost for each conversion
- Formula: Spent / Conversions

**ROI (Return on Investment)**
- Estimated return on ad spend
- Formula: ((Revenue - Spent) / Spent) Ã— 100
